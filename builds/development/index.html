Case 1 : Each row becomes a table

In this case, we've already figured out our minimum table width is about 760px so we'll set up our media query to take effect when the screen gets narrower than that.

We are going to force the table to not behave like a table by setting every table-related element to be block-level. It's kind of like each table row becomes a table in itself, but only as wide as the screen. No more horizontal scrolling!

Then for each "cell", we'll use CSS generated content (:before) to apply the label, so we know what each bit of data means.
